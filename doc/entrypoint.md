TangoMan Entrypoint
===

## ℹ️ Infos

- author:  "Matthias Morin" <mat@tangoman.io>
- version: 1.0.0-light
- link:    https://github.com/TangoMan75/shoe

## 📑 Description

Execute Development and Continuous Integration tasks.   

## 🔥 Usage

`sh entrypoint.sh [command] `

## 🤖 Commands

1. **`lint`**
  - Sniff errors with linter

2. **`tests`**
  - Run tests

3. **`git_hooks`**
  - Install git hooks

4. **`bash_unit`**
  - Install bash_unit framework

5. **`alpine`**
  - Run script in Alpine Docker container

6. **`busybox`**
  - Run script in Busybox Docker container

7. **`ubuntu`**
  - Run script in Ubuntu Docker container

8. **`help`**
  - Print help for current shoe script (default)
